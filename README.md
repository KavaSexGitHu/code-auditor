## 📖 What is code-auditor?

code-auditor is a tool that helps you find copied or duplicated logic in computer programs. It looks at code from different programming languages to spot where someone might have copied ideas or solutions. This can be useful if you want to check school assignments, work projects, or software for originality and security risks.

The tool uses smart methods to compare how code works, not just the exact words. This helps find hidden copying, even if the code looks different. It works on Windows computers with a simple interface.
